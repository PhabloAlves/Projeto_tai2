# TAI-2 Project

## Overview
TAI-2 is a comprehensive web application project built with modern web technologies. This full-stack application combines a robust backend with an intuitive frontend interface to deliver a complete solution for business management and system administration.

## Features
- **Responsive Web Interface**: User-friendly interface built with JavaScript, CSS, and HTML
- **Robust Backend**: PHP-based server-side logic for reliable data processing
- **Database Integration**: Efficient data management and storage
- **Modular Architecture**: Well-organized code structure with separate frontend and backend directories
- **Modern UI/UX**: Clean and intuitive user experience design
- **Cross-Platform Support**: Works seamlessly across different browsers and devices

## Technology Stack
- **Frontend**:
  - JavaScript (48.8%)
  - CSS (44.2%)
  - HTML/Blade Templates
- **Backend**:
  - PHP (4.7%)
  - Blade Template Engine (2.3%)
- **Architecture**: Full-stack web application with separate front_end and back_end directories

## Project Structure
```
Projeto_tai2/
├── back_end/          # Backend PHP code and server logic
```

## Installation & Setup

### Prerequisites
- PHP 7.0 or higher
- Node.js and npm (for frontend development)
- A modern web browser
- Web server (Apache, Nginx, or similar)

### Backend Setup
1. Navigate to the `back_end` directory
2. Configure your PHP environment and database connection
3. Set up any required PHP dependencies

### Frontend Setup
1. Navigate to the `front_end` directory
2. Install dependencies (if applicable):
   ```bash
   npm install
   ```
3. Build or compile frontend assets as needed

## Getting Started
1. Clone the repository
2. Set up both backend and frontend as described above
3. Configure your web server to serve the application
4. Access the application through your web browser

## Development

### Frontend Development
- Modify JavaScript files in the frontend directory

### Backend Development

- Implement business logic and data processing
- Maintain database schema and operations
